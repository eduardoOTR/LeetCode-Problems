# LeetCode Problem Solutions

This repository contains my solutions to [LeetCode](https://leetcode.com/) problems in different programming languages.  
It aims to showcase my progress, help others interested in the solutions, and improve my skills in Algorithms and Data Structures and programming in general.

## Repository Structure

- `/AED1/` - Evaluation exercises completed for Algorithms and Data Structures I.
- `/C/` - Solutions in C
- `/Python/` - Solutions in Python
